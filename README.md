## ADT란?

  - **추상 데이터 타입(ADT : Abstract Data Type)은**, **객체의 명세**와 이들 객체에 대한 **연산의 명세**가 **객체의 표현**과, **연산의 구현으로부터** **분리된 방식으로 구성**된 데이터 타입이다.
  - **명세와 구현의 구분**, 어떤 **시스템의 간략화된 기술**, 시스템의 **핵심적인 구조나 동작에만 집중**하는 것
